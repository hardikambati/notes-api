# Notes API

### Steps to setup:

1. Create & activate virtual env (for ubuntu)
```
    virtualenv -p python3 venv
    source venv/bin/activate
    cd main
```

2. Install requirements
```
    pip install -r requirements.txt
``` 

3. Run the server
```
    python3 manage.py runserver
```
